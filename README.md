# 🌦️ Weather App

A simple and responsive Weather Application built using HTML, CSS, and JavaScript. The app fetches real-time weather data using the OpenWeatherMap API.

## 🚀 Features

- Search weather by city name
- Real-time temperature updates
- Displays humidity and weather conditions
- Responsive user interface
- Easy-to-use design

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- OpenWeatherMap API

## 📂 Project Structure

weatherapp/
├── index.html
├── style.css
├── script.js
└── README.md

## ⚙️ How to Run

1. Download or clone the repository.
2. Open `index.html` in your browser.
3. Enter a city name.
4. Click the Search button to view weather details.
